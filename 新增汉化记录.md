## 有意未翻译的内容

以下字符串因与 Monaco 编辑器内部代码冲突而**故意跳过**：

| 类别 | 示例 | 原因 |
|------|------|------|
| 差异对比 | `Accept`、`Reject` | Monaco 内联编辑工具栏 |
| 编辑导航 | `Jump Back`、`Jump To Edit` | Monaco 跳转功能 |
| 代码折叠 | `Click to expand` | Monaco 折叠展开 |
| 命令面板 | `Open folder…`、`Find and Replace` | Monaco 命令 |
| 键盘按键 | `Alt`、`Shift`、`Ctrl`、`Esc`、`Tab`、`Enter` | 键位映射（与 Monaco 共享） |
| 语法/着色 | 各种高亮描述字符串 | Monaco 语法引擎 |
| 折叠/部件 | `folding-*`、` zone-widget-*`、`snippet-*` | Monaco 视觉装饰 |
| 跟踪范围 | `tracked-range-*` | Monaco 内部状态 |
| 代码参考 | `goto-definition-link`、`codelens` | Monaco 导航功能 |
| CLI 命令 | `claude mcp add` | 保留原样（命令即源码） |
| 模型名称 | `Fable 5` | 产品名称不翻译 |
